# Lok-Sabha-Members-Attendance-Analysis
This project analyzes attendance patterns of Members of Parliament (MPs) in the Lok Sabha using data-driven techniques. The goal is to uncover trends, identify inconsistencies, and provide meaningful insights through data cleaning, analysis, and visualization.
# Project Overview
Collected and processed Lok Sabha attendance dataset
Cleaned and transformed raw data using Python (Pandas)
Performed Exploratory Data Analysis (EDA) using SQL
Built an interactive Power BI dashboard to visualize attendance trends
# Tech Stack
Programming Language: Python
Libraries: Pandas, NumPy
Database / Querying: SQL
# Key Insights
Attendance trends clearly highlight decline or improvement patterns over sessions
Visual dashboards make it easy to identify underperforming MPs
Data-driven insights can help in policy evaluation and accountability
Comparative analysis across parties reveals participation disparities
